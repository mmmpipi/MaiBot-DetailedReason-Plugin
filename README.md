# 麦麦思考插件

帮助推理能力弱的模型更好地对逻辑内容进行推理，相比于thinking模型更快，且推理是可选的
使用此插件时最好将replyer模型或你将使用的模型设置一个token上限
如果你的模型很强那么这个插件可能不会帮到你

## 配置说明

### 插件信息 `[plugin]`

- `enabled`：是否启用插件（默认 true）
- `version`：插件版本
- `config_version`：配置文件版本
- `use_model`：使用的模型（默认 `replyer`）
